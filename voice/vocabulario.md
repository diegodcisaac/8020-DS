# Vocabulário: evitar e usar

Lista viva de substituições recorrentes. A coluna da direita não é tradução mecânica: escolhe-se a forma pela função real da frase.

## Anglicismos

| Evitar | Usar |
|---|---|
| governance | governança |
| tracking | medição, acompanhamento |
| copy | texto editorial |
| callcenter | central de atendimento |
| A/B test | teste de variantes |
| venue, venues | espaço, endereço, centro, ou omitir |
| always-on | contínuo, permanente |
| owner | responsável |
| stack | conjunto, pilha |
| insight (em excesso) | leitura, achado |

## Vícios de linguagem

| Evitar | Usar |
|---|---|
| "real" como intensificador (risco real, conversas reais) | remover; se precisar, principal, central, crítico |
| "nomear" como tique | declarar, registrar, identificar, apontar, expor, definir, descrever |
| paisagem (em uso editorial) | cenário |
| mandato (formal, declarado) | aprovação, autoridade declarada, responsabilidade, papel |
| vazio (para território disponível) | disponível, em aberto |
| salência | presença na cabeça do público, presença reconhecida |
| escrutínio | avaliar |
| autoelogio (leitura honesta, documento forte) | neutro factual |

## Jargão técnico em peça cliente

| Evitar | Usar |
|---|---|
| atos | fases |
| usuário pendular | viajante de longa distância |
| Living / Looking / Buying | Conviver / Avaliar / Aceitar |
| o vizinho (vago) | nomear quem é (São Paulo, Paraguai, Bolívia) |
| "Spotify Investimentos brasileiro" (justaposição canal e tema) | "podcasts de investimentos no Spotify brasileiro" |

## Forma e pontuação

| Evitar | Usar |
|---|---|
| travessão (—) | dois-pontos, vírgula, parênteses ou ponto |
| abreviação (7d, h, min) | por extenso (sete dias úteis, horas, minutos) |
| sigla na primeira menção | nome por extenso (Banco do Brasil) |
| título que conta itens (Os seis segmentos) | título que atiça (o assunto, não a contagem) |
| autorreferência (este documento mostra) | liderar pelo assunto |
| frase de slogan em parecer (X chegou, Y assenta) | prosa formal corrida |
